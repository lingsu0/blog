# 哈希

## 1. 两数之和

[原题链接](https://leetcode.cn/problems/two-sum/)


遍历数组，对于每个数字 `nums[i]`，先检查哈希表中是否存在 `target - nums[i]`。如果存在，说明找到了两个数之和等于目标值，直接返回它们的下标；如果不存在，则将当前数字及其下标存入哈希表。这样既能快速查找，又能避免数字与自己匹配。

```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        // 创建一个哈希表
        Map<Integer , Integer> numtable = new HashMap<Integer , Integer>();
        int n = nums.length;
        for(int i = 0 ; i < n ; ++i){
            // 判断当前数字是否在哈希表中
            if(numtable.containsKey(target - nums[i])){
                // 如果在哈希表中，则返回结果
                return new int[]{numtable.get(target - nums[i]), i};
            }
            // 否则将当前数字加入哈希表
            numtable.put(nums[i],i);
        }
        // 如果没有找到结果，则返回空数组
        return new int[0];
    }
}
```

## 49. 字母异位词分组

[原题链接](https://leetcode.cn/problems/group-anagrams/)

创建一个哈希表，将每个字符串排序并作为键，将原字符串作为值。这样，相同的字母异位词会被归为一组，并返回哈希表中的值。

```java
class Solution {
    public List<List<String>> groupAnagrams(String[] strs) {
        // 创建一个哈希表
        Map<String, List<String>> map = new HashMap<String, List<String>>();
        // 对每个字符串进行排序
        for(String str : strs){
            // 将字符串转换为字符数组
            char[] arr = str.toCharArray();
            // 对字符数组进行排序
            Arrays.sort(arr);
            // 将排序后的字符串作为键
            String key = new String(arr);
            // 获取键对应的值，如果键不存在，则创建一个空列表
            List<String> list = map.getOrDefault(key,new ArrayList<String>());
            //将字符串添加到列表中
            list.add(str);
            // 将列表添加到哈希表中
            map.put(key , list);
        }
        // 将哈希表中的值转换为列表并返回
        return new ArrayList<List<String>>(map.values());
    }
}
```

## 128. 最长连续序列

[原题链接](https://leetcode.cn/problems/longest-consecutive-sequence/)

首先创建一个哈希集合，遍历输入数组，将所有元素存入集合中以实现 O(1) 时间的查找效率并自动去重。随后再次遍历集合中的每个元素，为了优化时间复杂度，仅当当前元素 `x` 的前一个数 `x-1` 不存在于集合中时（说明 `x` 是某个连续序列的起点），才从 `x` 开始向后枚举 `x+1`, `x+2`... 直到找不到连续数字为止。在此过程中记录并更新遇到的最长连续序列长度。

```java
class Solution {
    public int longestConsecutive(int[] nums) {
        // 创建一个集合
        Set<Integer> ss = new HashSet<>();
        // 将数组中的元素添加到集合中
        for(int num : nums){
            ss.add(num);
        }
        // 遍历集合中的元素
        int res = 0;
        for(int x : ss){
            // 如果当前元素前面有元素，则跳过
            if(ss.contains(x-1)) continue;
            // 否则，从当前元素开始，找到连续的元素
            int y = x + 1;
            // 找到连续的元素
            while(ss.contains(y)) y++;
            // 更新最长连续序列长度
            res = Math.max(res , y - x);
        }
        return res;
    }
}
```

## 283. 移动零


[原题链接](https://leetcode.cn/problems/move-zeroes/)

创建两个指针 `i` 和 `j`，初始时 `i` 指向第一个非零元素的位置，`j` 指向当前遍历到的位置。遍历数组，如果当前元素不为零，则将当前元素与 `nums[i]` 交换，并同时将 `i` 和 `j` 向后移动一位。这样，数组中 `i` 到 `j` 之间的元素就是非零元素。

```java
class Solution {
    public void moveZeroes(int[] nums) {
        int n = nums.length;
        int j = 0;
        int i = 0;
        while(j < n){
            if(nums[j] != 0){
                int temp = nums[i];
                nums[i] = nums[j];
                nums[j] = temp;
                i++;
            }
            j++;
        }
    }
}
```
## 11. 盛最多水的容器

[原题链接](https://leetcode.cn/problems/container-with-most-water/)

使用双指针法，初始化左指针 `left` 指向数组起始位置，右指针 `right` 指向数组末尾。计算当前由左右指针构成的容器面积（高度取两边较小值，宽度为两指针距离），并更新最大面积。为了寻找更大的面积，移动高度较小的那一侧指针（因为移动较高一侧只会让宽度变小且高度受限于较短边，面积必然减小或不变；而移动较短一侧虽然宽度变小，但可能遇到更高的边从而增大面积）。重复此过程直到两指针相遇。

```java
class Solution {
    public int maxArea(int[] height) {
        int n = height.length, left = 0 , right = n - 1;
        int res = 0;
        while(left < right){
            int result = Math.min(height[left],height[right]) * (right-left);
            res = Math.max(res , result);
            if(height[right] >= height[left]) left++;
            else right--; 
        }
        return res;
    }
}
```

## 15.三数之和

[原题链接](https://leetcode.cn/problems/3sum/)

创建一个结果列表 `res`，对输入数组进行排序，并使用三指针法解决。外层循环迭代数组中的每个元素，内层循环迭代从当前元素之后的元素。在内层循环中，使用双指针法寻找满足条件的三数之和。

```java
class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        List<List<Integer>> res = new ArrayList();
        Arrays.sort(nums);
        int n = nums.length;
        for(int i = 0 ; i < n ; i++){
            if(i > 0 && nums[i] == nums[i-1]) continue;
            int num = -nums[i];
            int left = i+1;
            int right = n-1;
            while(left < right){
                if(nums[left] + nums[right] == num){
                    res.add(Arrays.asList(nums[i],nums[left],nums[right]));
                    while(left+1 < right && nums[left] == nums[left+1]) left++;
                    while(right-1 < left && nums[right] == nums[right-1]) right--;
                    left++;
                    right--;
                }
                else if(nums[right] + nums[left] > num ) right--;
                else if(nums[right] + nums[left] < num ) left++;
            }

        }
        return res;
    }
}
```

## 42.接雨水

[原题链接](https://leetcode.cn/problems/trapping-rain-water/)

使用单调栈解决，创建一个栈，遍历数组，如果当前元素小于栈顶元素，则将当前元素入栈。如果当前元素大于栈顶元素，则将栈顶元素出栈，并计算当前栈顶元素与栈中前一个元素之间的宽度乘以栈顶元素与当前元素之间的高度，并累加到结果中。重复此过程直到栈为空或者当前元素小于栈顶元素。

```java
class Solution {
    public int trap(int[] height) {
        int n = height.length;
        Deque<Integer> stack = new LinkedList<Integer>();
        int res = 0;
        for(int i = 0 ; i < n ; i++){
            while(!stack.isEmpty() && height[i] > height[stack.peek()]){
                int pop = stack.pop();
                if(stack.isEmpty()) break;
                int left = stack.peek();
                int Width = i - left - 1;
                int Height = Math.min(height[left] , height[i] ) - height[pop];
                res += Width * Height;
            }
            stack.push(i);
        }
        return res;
    }
}
```
